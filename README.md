
# 🛍️ AI Product Comparison 

An advanced AI-powered product comparison tool that evaluates multiple products based on price, features, and value, and provides intelligent recommendations using CrewAI and Gemini.

---

## 🚀 Features

- 🛒 Multi-product comparison (2–5 products)
- 📊 Price comparison dashboard
- ⚖️ Feature-based scoring system
- 💰 Value-for-money analysis
- 🧠 AI-powered deep comparison (CrewAI)
- 🏆 Best product recommendation
- 📈 Visualization (bar charts)
- 📥 Downloadable comparison report

---

## 🛠️ Tech Stack

- Python
- Streamlit
- CrewAI (Multi-Agent AI)
- Google Gemini API
- Pandas
- Matplotlib

---

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-product-comparison.git
cd ai-product-comparison
````

---

### 2️⃣ Install Dependencies

```bash id="install-cmd-001"
pip install -r requirements.txt
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file:

```env id="env-prod-001"
GOOGLE_API_KEY=your_gemini_api_key
```

Get your API key:
👉 [https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)

---

### 4️⃣ Run the App

```bash id="run-cmd-001"
streamlit run app.py
```

---

## 🧪 How to Use

1. Select number of products (2–5)
2. Enter:

   * Product names
   * Prices
   * Features (comma-separated)
3. View:

   * Comparison table 📊
   * Price chart 📈
   * Value scores ⚖️
4. Click **AI Deep Comparison**
5. Get:

   * Feature comparison
   * Pros & cons
   * Best product recommendation 🏆
6. Download report 📥
